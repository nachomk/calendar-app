⚙️ Configuración inicial

Para que los datos se sincronicen entre todos tus dispositivos, necesitás conectar la app a tu base de datos de Firebase. Solo se hace una vez.

  1. Andá a console.firebase.google.com
  2. Creá un proyecto nuevo (cualquier nombre)
  3. Entrá al proyecto → Firestore Database → Crear base de datos → Modo de prueba
  4. Andá a Configuración del proyecto (ícono ⚙️) → Tus apps → Agregar app → Web (</>)
  5. Copiá los valores del firebaseConfig y pegálos abajo
No se pudo conectar. Revisá los datos ingresados.
